# Documentation: Roles

There are multiple types of roles.

All roles share the same attributes, but will have different values for each.

## Role Attributes

Every role contains (at minimum) the following attributes:

- Title
- Purpose
- Responsibilities
- Domain
- Statistics
- Solo / Team
- Members (team role only)
