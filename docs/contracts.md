# Documentation: Contracts

There are multiple types of roles.

All roles share the same attributes, but will have different values for each.

## Contract Attributes

Every contract contains (at minimum) the following attributes:

- Name
- Purpose
- Values
- Terms
  - Commitments
  - Success criteria
- Assessment method
- Statistics
